This repository is my personal roadmap for learning bug bounty hunting step by step.

## Goal
- Build strong web security fundamentals.
- Practice common vulnerabilities in labs.
- Learn real-world bug bounty methodology.
- Improve reporting, recon, and manual testing skills.

## What I need to learn

### 1. Web Fundamentals
- How the web works
- HTTP/HTTPS
- Requests and responses
- Cookies, sessions, headers
- DNS basics
- Browser behavior

### 2. Networking Basics
- TCP/IP
- Ports and services
- Client-server model
- Basic packet understanding

### 3. Operating Systems
- Linux commands
- Windows basics
- File systems
- Permissions
- Processes and networking tools

### 4. Web Technologies
- HTML
- CSS
- JavaScript
- Backend basics
- APIs
- JSON
- Authentication and authorization

### 5. Programming Skills
- Python for automation
- Basic Bash scripting
- JavaScript for web understanding
- Reading code and spotting logic mistakes

### 6. Security Concepts
- OWASP Top 10
- IDOR
- XSS
- SQL Injection
- CSRF
- SSRF
- File upload bugs
- Open redirect
- Broken authentication
- Broken access control

### 7. Bug Bounty Methodology
- Scope reading
- Asset prioritization
- Recon basics
- Manual testing
- Note taking
- Reporting
- Retesting and verification
